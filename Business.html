РЕЗЮМЕ ПРОЕКТА

Sakha-China — многоплатформенный маркетплейс-агрегатор для импорта товаров из Китая. Интеграция с Poizon, 1688, Taobao, Alibaba, прямые контракты с заводами-производителями. B2C розница + B2B опт + контрактное производство. Полный цикл: поиск поставщика → выкуп → логистика → таможня → доставка клиенту.

География

Якутск → Дальний Восток → Сибирь → РФ → СНГ

Целевая аудитория

B2C: 70 000 чел. (Якутск) | B2B: 5 000+ компаний (ДВ)

Рынок

Импорт КНР→РФ: $100 млрд/год | Рост: 15-20%/год

БИЗНЕС-МОДЕЛЬ (CANVAS)

Сегменты клиентов

  • B2C Розница: Молодёжь 18-35 (брендовая одежда, обувь, электроника)
  • B2C Премиум: Коллекционеры (лимитированные серии, ретро-модели)
  • B2B Малый бизнес: Магазины одежды, реселлеры, Instagram-магазины
  • B2B Средний бизнес: Региональные ритейлеры (ДВ, Сибирь)
  • B2B Крупный бизнес: Закупщики оборудования, стройматериалов, запчастей
  • B2B Производство: Контрактное изготовление под собственным брендом (OEM/ODM)

Ценностные предложения

  • Единая платформа для всех китайских площадок (Poizon, 1688, Taobao, Alibaba)
  • Гарантия подлинности через официальную аутентификацию Poizon
  • Прямые контракты с заводами-производителями (без посредников)
  • Полный цикл импорта: поиск → выкуп → логистика → таможня → доставка
  • Оптимизация затрат: сравнение вариантов доставки (агрегатор)
  • Документальное сопровождение (сертификаты, декларации, маркировка)
  • Контрактное производство: разработка дизайна, изготовление, брендирование
  • Оплата в рублях, поддержка на русском языке, юридическая защита

Каналы продаж

  • Мобильное приложение (iOS / Android)
  • Веб-платформа (адаптивная версия)
  • Telegram-бот (быстрые заказы, уведомления)
  • B2B-портал (оптовые заказы, контрактное производство)
  • Локальный маркетинг (Якутск, ДВ)
  • Реферальная программа (клиенты привлекают клиентов)
  • Партнёрская сеть (региональные представители)

Взаимоотношения с клиентами

  • Персональный менеджер для B2B-клиентов
  • Премиум-подписка для B2C (приоритет, кэшбэк, персональная поддержка)
  • Автоматизированная поддержка (чат-бот, база знаний)
  • Программа лояльности (баллы за заказы, отзывы, рефералов)
  • Гарантия возврата при несоответствии товара
  • Прозрачный трекинг на всех этапах

Потоки доходов

  • Розничная наценка: 25-35% на товары из каталога
  • Комиссия агрегатора: 10-20% от стоимости доставки (партнёры)
  • Комиссия за поиск: 5-10% от стоимости кастомного заказа
  • B2B опт: 15-25% при объёме от 50 единиц
  • Контрактное производство: 20-30% от стоимости партии
  • Подписка: 299-999 ₽/мес (премиум-функции)
  • Платное размещение: 5 000-20 000 ₽/мес (поставщики)
  • Страхование грузов: 2-5% комиссии от стоимости

Ключевые ресурсы

  • Юридические лица: ООО (РФ), WFOE (Китай)

  • Команда: разработка, операции, поддержка, логистика
  • Финансовые ресурсы: оборотные средства, кредитные линии

Ключевые виды деятельности

  • Разработка и поддержка IT-платформы
  • Поиск и верификация поставщиков в Китае
  • Управление логистическими цепочками
  • Таможенное оформление и сертификация
  • Маркетинг и привлечение клиентов

  • Управление финансовыми потоками (конвертация, платежи)

Ключевые партнёры

  • Площадки: Poizon, 1688, Taobao, Alibaba (официальные API)
  • Заводы-производители: прямые контракты (OEM/ODM)
  • Логистика: карго-компании, авиа-, авто-, ж/д перевозчики
  • Таможенные брокеры: Благовещенск, Москва
  • Платёжные агенты: конвертация RUB→CNY

  • Сертификационные центры: «Честный ЗНАК», Росаккредитация

Структура расходов

  • Закупка товара: 45-55% от выручки
  • Логистика: 15-20% (Китай → РФ → клиент)
  • Таможенные платежи: 18-22% (пошлина + НДС)
  • Комиссии площадок: 5-8% (Poizon, 1688, Alibaba)
  • Финансовые комиссии: 3-5% (конвертация, эквайринг)
  • OPEX: 8-12% (З/п, офис, IT, маркетинг)
  • Налоги: 15% УСН (прибыль)

ИНТЕГРАЦИЯ С ПЛОЩАДКАМИ

Poizon (Dewu)

  • Категории: Обувь, одежда, аксессуары, электроника
  • Уникальность: Официальная аутентификация (100% оригинал)
  • Интеграция: Официальный API (партнёрство через WFOE)
  • Комиссия: 6% + 20 CNY фикс за аутентификацию
  • Срок: 1-2 дня проверка + доставка
  • Целевая аудитория: B2C премиум, коллекционеры

1688.com

  • Категории: Опт, одежда, электроника, товары для дома
  • Уникальность: Внутренний рынок Китая (ниже цены чем Alibaba)
  • Интеграция: API через агрегаторов / байеров
  • Комиссия: 3-5% через посредников
  • Мин. заказ: От 2-3 единиц (зависит от продавца)
  • Целевая аудитория: B2B малый бизнес, реселлеры

Taobao

  • Категории: Розница, одежда, аксессуары, товары для дома
  • Уникальность: Огромный ассортимент, низкие цены
  • Интеграция: API через агрегаторов
  • Комиссия: 5-8% через посредников
  • Мин. заказ: 1 единица
  • Целевая аудитория: B2C розница

Alibaba.com

  • Категории: Опт, оборудование, сырьё, комплектующие
  • Уникальность: B2B экспортная площадка (англоязычная)
  • Интеграция: Официальный API (партнёрство)
  • Комиссия: 2-5% (зависит от категории)
  • Мин. заказ: От 50-100 единиц (опт)
  • Целевая аудитория: B2B средний и крупный бизнес

Прямые заводы

  • Категории: Контрактное производство (OEM/ODM)
  • Уникальность: Без посредников, кастомизация, брендирование
  • Интеграция: Прямые контракты (юридические соглашения)
  • Комиссия: 0% (прямая оплата заводу)
  • Мин. заказ: От 500-1000 единиц (зависит от завода)
  • Целевая аудитория: B2B крупный бизнес, собственные бренды
